# Netflix-data-analysis-using-power-BI
This project focuses on analyzing Netflix movies and TV shows to uncover insights about content distribution, genre trends, ratings, release patterns, and audience preferences. The aim is to transform raw Netflix data into meaningful visualizations using Power BI, helping stakeholders understand the type of content that performs best across different countries, age groups, and categories.
